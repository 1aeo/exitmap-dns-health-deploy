This service monitors DNS resolution capability across all Tor exit relays. Each relay is tested by resolving a unique wildcard subdomain and verifying the expected IP is returned.

Validation Method:

  • Wildcard DNS: Query unique subdomain (e.g., {uuid}.{fingerprint}.tor.exit.validator.1aeo.com)
  • IP Verification: Confirm resolution returns expected IP address
  • Error Detection: Identify NXDOMAIN, timeouts, wrong IPs, and SOCKS errors
